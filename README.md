# Translator-Tkinter
* An app made using python that translates your text into any desired language.
## Technologies Used
* Python3 
* Libraries Used
   * tkinter
   * googletrans
## Functionality
* This translator can help translate dozens of languages.
* Translation is allowed only through text/typing into the application.
